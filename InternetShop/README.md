# Интернет-магазин (учебный проект)

Проект создан в Visual Studio 2022 с использованием пустого шаблона ASP.NET Core.

## Описание проекта

Это учебный проект для будущего интернет-магазина. На данном этапе реализована базовая структура с несколькими страницами и контроллером для демонстрации работы MVC.

## Функциональность

В проекте реализовано:
- контроллер `HomeController` с двумя действиями (actions)
- две страницы (представления):
  - **главная страница (Index)** — приветствие и описание магазина;
  - **страница с товарами (Products)** — статический список товаров, заполненный прямо во View.
